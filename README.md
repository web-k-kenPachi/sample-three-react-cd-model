## React-three-fiber で３Dモデルを描画したサイト
React-three-fiberで3Dモデルを表示しております。
スクロールと連動したアニメーションはGsapのScrollTriggerで対応しております。

## 主なプラグイン
- react
- three.js
- react-three-fiber
- gsap
- framer-motion
- swiper

## 起動
```
npm i
npm run dev
```
